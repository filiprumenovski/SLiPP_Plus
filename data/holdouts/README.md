# Holdout ID Lists

These CSVs are derived directly from the supplementary spreadsheets in `data/raw/supplementary/`:

- `apo_pdb_ids.csv`: 117 unique `pdb_id` values from `ci5c01076_si_003.xlsx`, sheet `Table S2`, parsed with row 2 as the header.
- `alphafold_ids.csv`: 149 unique `uniprot_id` values from `ci5c01076_si_004.xlsx`, sheet `Table S3`, parsed with row 2 as the header.

The counts above are the counts present in this checkout's source workbooks. They intentionally do not use older handoff counts that do not match the files currently in the repository.
