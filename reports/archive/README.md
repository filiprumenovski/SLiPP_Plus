# reports/archive/

Per-experiment report directories and stale top-level summaries that are
no longer load-bearing for the current submission. Preserved for the
research audit trail; not referenced by any active code path.

Read [`reports/ablation_matrix.md`](../ablation_matrix.md) for the
machine-generated index of every experiment, including the ones in here.
Each row links back to the originating registry entry in
[`experiments/registry.yaml`](../../experiments/registry.yaml).

The currently load-bearing report dirs live one level up:

- [`compact_publishable/`](../compact_publishable/) — paper ablation ladder.
- [`compact_subset_ensemble_sweep/`](../compact_subset_ensemble_sweep/) — exp-021 (deployable).
- [`compact_shape6_shell6shape3_hydro4_geom_chem_ensemble/`](../compact_shape6_shell6shape3_hydro4_geom_chem_ensemble/) — exp-019 (internal best).
- [`publication/`](../publication/) — submission figure set.
